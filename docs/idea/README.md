# TPO Coach — 아이디어 개요

> 상태: **아이디어/기획 단계**. 아래 내용은 주제와 대략적인 플로우를 정리한 것으로,
> 도구(tool) 스키마·API·데이터 모델은 아직 확정 전이다.

## TPO Coach 란

**TPO**(Time·Place·Occasion, 시간·장소·상황)에 맞는 옷차림을 코칭해 주는 MCP 서버.
"이 자리에 뭘 입고 가야 하지?" 라는 흔한 고민을, 상황 설명만으로 **권장 드레스코드**와
**구체적인 착장 가이드**로 바꿔 준다.

핵심 문제의식:

- 정장은 과하고 평소 옷은 애매한 **"적정선"** 을 찾기 어렵다.
- 모임마다 암묵적인 규칙(하객룩 색상, 골프장 복장 규정 등)이 달라 실수하기 쉽다.
- 여러 명이 모이는 자리에서는 **참석자 간 눈높이**를 맞추기가 더 어렵다.

## 착장 추천 플로우 (개인)

개인이 착장을 맞출 때의 대략적인 흐름:

1. **의상 색상 뽑기** — 사진/입력에서 기준 의상의 색상을 추출
2. **어울리는 하의 찾기** — 데이터(셋업 기반)에서 색상과 어울리는 하의 후보 매칭
3. **유사도 높은 하의 보여주기** — 후보를 유사도 순으로 추천

자세한 내용은 [matching-flow.md](matching-flow.md) 참고.

## 활용 시나리오

네트워킹 행사, 소개팅, 해커톤, 결혼식 하객룩, 골프장 라운딩, 학교 행사 등.
전체 목록과 각 상황의 주의점은 [use-cases.md](use-cases.md) 참고.

## 확장 포인트

- **톡캘린더 연동** — 일정에 등록된 모임 정보(시간/장소)를 가져와 가이드 자동 생성.
- **개인 옷장 데이터** — 보유 의상을 등록해 두면 가이드에 맞는 조합을 바로 추천.
