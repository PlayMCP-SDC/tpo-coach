# 착장 추천 플로우 (색상 기반)

> 상태: **1차 메모 · v1 연기**. 이 색상 매칭 플로우(F8/F9)는 C 범위에서 **v1**로 미뤘다 — MVP 는 텍스트 기반
> `check_outfit`. 범위·우선순위는 [features.md](features.md)·[delivery-plan.md](delivery-plan.md) 참고.

개인이 착장을 맞출 때, 기준 의상의 **색상**에서 출발해 어울리는 하의를 찾아 주는
대략적인 흐름. (세부 알고리즘·데이터 스키마는 확정 전)

## 3단계 흐름

```
[기준 의상]
   │  ① 색상 추출
   ▼
[대표 색상]
   │  ② 셋업 데이터에서 색 매칭
   ▼
[어울리는 하의 후보]
   │  ③ 유사도 정렬
   ▼
[추천 하의 Top-N]
```

### ① 의상 색상 뽑기

- 기준 의상의 색/설명 텍스트(기본) 또는 `image_url`(보조)에서 **대표 색상**을 추출.
- 단일 색 또는 주요 색 팔레트 형태로 표현.

### ② 색상과 어울리는 하의 찾기

- 데이터로 받은 **셋업(상·하의 조합) 기반**으로, 추출한 색상과 어울리는 하의 후보를 찾는다.
- "어울림"의 기준(보색/톤온톤/무채색 매칭 등)은 추후 규칙/데이터로 정의.

### ③ 유사도 높은 하의 보여주기

- 후보 하의를 **유사도 순으로 정렬**해 상위 N개를 추천.
- 모임장 모드의 가이드(권장 수준·금기)와 결합하면, 그 자리에 맞으면서 색도 어울리는
  하의를 우선 노출할 수 있다.
