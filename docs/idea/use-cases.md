# 활용 시나리오

모임장 모드(와 개인 착장 검사)가 특히 유용한 상황들. 공통점은 **"적정선 찾기가 애매한"**
자리라는 점 — 정장은 과하고 평소 옷은 부족한, 그 사이를 잡아 주는 게 핵심 가치다.

| 상황 | 어려운 점 | TPO Coach 가 잡아 주는 것 |
| --- | --- | --- |
| 교육 프로그램 네트워킹 행사 / 대학 IT 동아리 면접 | 정장은 딱딱하고 평소 옷은 애매함 | **단정한 캐주얼**의 적정선 |
| 소개팅 · 블라인드 데이트 | 과하게 꾸미지 않으면서 호감 주기 | **꾸안꾸** 스타일링 |
| 해커톤 · 팀 프로젝트 워크숍 | 멘토/외부 인사 발표 자리가 섞임 | **편안함 + 최소한의 격식** 균형 |
| 결혼식 (하객룩) | 튀는 색상, 특히 **흰색** 금기 | 하객 매너에 맞는 색·아이템 |
| 골프장 (라운딩) | 보수적 CC 는 카라 없는 셔츠·반바지·청바지 금지 | 골프장 **복장 규정** 반영 |
| 학교 행사 등 | 행사 성격별로 기대치가 다름 | 상황별 권장 수준 |

## 시나리오별 메모

- **네트워킹 / 면접** — "단정한 캐주얼"의 기준이 사람마다 달라 참석자 눈높이를 맞추기 어렵다.
  모임장 카드로 기준선을 공유하면 편차가 줄어든다.
- **소개팅** — 너무 꾸미면 부담, 안 꾸미면 성의 없어 보이는 양극단 사이의 "꾸안꾸"가 목표.
- **해커톤 / 워크숍** — 평상시 개발 복장과, 외부 인사 발표 때의 복장 사이를 구분해 안내.
- **결혼식 하객룩** — 신부와 겹치는 흰색/아이보리, 과한 노출·드레스 등 금기 위주로 가드.
- **골프장** — CC 마다 규정이 다르므로, 보수적 기준을 기본값으로 두고 안내하는 편이 안전.

> 위 목록은 초기 후보군이다. 시나리오마다 "권장 수준"과 "금기 항목"의 규칙 세트가 달라지므로,
> 구현 시 상황 → 규칙 매핑을 데이터로 분리해 관리하는 것을 검토한다.
